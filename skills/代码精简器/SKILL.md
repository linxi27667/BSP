# 代码精简器

本 skill 包含一个代码简化代理，用于优化代码质量和可维护性。

## 使用方式

当用户完成代码编写或修改后，spawn 一个子代理来简化代码：

1. 读取 `agents/code-simplifier.md` 中的完整指令
2. 让子代理分析最近修改的代码
3. 应用优化建议，保持功能不变

## 代理指令

完整的代理指令在 `agents/code-simplifier.md` 中。
